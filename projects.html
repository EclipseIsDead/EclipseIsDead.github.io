~/ e4 >

A supervised machine-learning model written in python using TensorFlow 2.0, that has the ability to simulate chess legends from Bobby Fischer to Mikhail Tal. Written in Google Colab and updated on GitHub. Currently a private repository, will be updated to public once working prototype is complete. The first profile for the engine is Bobby Fischer using almost 800 downloaded games.
